# 엔터테이너 에이전트 RUNBOOK (v1)
모드: YouTube (롱폼/쇼츠), 팟캐스트, 라이브

## 0. 점화
- 상태: 행동의 불 ON → 15분 내 1개 퍼블리시.
- 타이머: 15분 샷건. 미완이라도 업로드.

## 1. 아이디어 캡처 (3분)
- 포맷 선택: [롱폼/쇼츠/팟캐스트/라이브]
- 훅 1문장, 약속 1문장, 로드맵 3불릿.

## 2. 대본 최소형 (5분)
- HPRC: Hook → Promise → Roadmap → Close
- 말하기 속도 x1.1, 문장 평균 12자.

## 3. 녹화 (3~7분)
- 휴대폰 1080p/30fps, 전면 카메라.
- 마이크: 라발리어 또는 스마트폰 내장, 입-마이크 15cm.
- OBS용 장면: Cam + BGM(-28 LUFS) + 로워서드.

## 4. 편집 (5~15분)
- 하드컷, 자막 자동(YouTube), 소프트 BGM.
- 쇼츠는 58초 이하, 자막 키워드 3개 강조.

## 5. 퍼블리시 (즉시)
- 제목 포뮬러: [문제/욕망] + [결과 약속] + [구체 수치/시간]
- 설명 템플릿 사용, 챕터 추가, 고정댓글에 CTA.

## 6. 배포 (5분)
- 커뮤니티 탭 요약 1문장 + 썸네일.
- 팟캐스트는 동일 오디오로 동시 송출.

## 7. 로그 & ΔI
- 아래 커맨드로 로그 기록:
  python chat_search.py --index chats --db index.pkl
  python chat_search.py --search "미러 엔진 ΔI 로그" --db index.pkl --topk 10
- mirror_card_template.json 업데이트 후 커밋.
